# client

- user processes
  - send query to db server
  - get some data

# server

- server processes
  - push query to PGA
- PGA / Program Global Area / Private Global Area
  - memory cache
  - all user has private PGA
- SGA / Shared Global Area
- Shared SQL Area
  - memory cache
  - to handle same queries from multiple users
  - keep `Execution Plan` to reuse
